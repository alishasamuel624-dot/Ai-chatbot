# Supported Intents

## Intents included
- greeting
- goodbye
- help
- services
- pricing
- hours
- contact
- thanks
- about
- fallback

## Example triggers
- greeting: hello, hi, hey
- goodbye: bye, see you later
- help: help, can you help
- services: what services do you offer
- pricing: how much does it cost
- hours: when are you open
- contact: how can I contact you
- thanks: thank you
- about: who are you
- fallback: unknown or unmatched input
